# Portfolio Chat Application

A modern portfolio chat application built with Next.js, featuring an AI-powered chatbot interface.

## Features

- AI-powered chat interface
- Portfolio information display
- Responsive design

### Environment Variables

Make sure to set the following environment variables in your Vercel dashboard:

- `GOOGLE_GENERATIVE_AI_API_KEY` - Your Google AI API key

### Build

```bash
npm install
```

### Development

```bash
npm run dev
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- AI SDK
